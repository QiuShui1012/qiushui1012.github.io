# No Infinite Effects in HUD

No Infinite Effects in HUD（Nieih）是一个仅客户端模组，会禁用主 HUD 右上角部分状态效果的显示。

状态效果是否被禁用取决于它们的持续时长与用户配置。

默认情况下，此模组只会禁用无限时长的效果。你可以根据时长是否无限来禁用效果，也可以根据效果的分类来禁用效果，甚至禁用整个模组。

这些配置在游戏内都可以修改。
